任务表

任务ID string
任务发布者ID  string
任务报酬 double
任务发布时间 long
任务类型（0、一直发，1、某个时间点发，2、间隔时间发）
任务脚本（Json序列化后的List<TaskShell>）  MEDIUMTEXT
任务状态 int (0、规划中，1、报名中，2、测试中，3、暂停，4、结束)
任务开始时间（可以定时开始，也可以手动开始）long
任务结束时间（有一个默认结束时间，也可以手动开始） long
任务持续时间  long
任务测试结果（Json格式化后的TaskResult）  MEDIUMTEXT
TaskShell
Tips:http协议就是请求类型+ip+端口+url+参数头+参数，tcp协议ip+端口+请求体（暂时先只支持这种）
{编号，IP，端口，协议类型，url，前置条件，请求类型，参数头<Key,Value>，参数<Key,Value>，请求体，预计结果(结果格式、字段、目标值)}

TaskResult
{
{总测试时长，总测试条目数，总成功条数，总失败条数，总请求失败数量，总预计结果比对失败数量}
{测试者ID，开始时间，结束时间，持续时间，测试条目数，成功数量，失败数量，请求失败数量，预计结果比对失败数量，失败消息体}
}
